Pythagoras Aufgabe 27

In einem rechtwinkligen Dreieck ist die Höhe das n-fache des Hypotenusenabschnitts q.  
Drücken Sie die Länge der Hypotenuse c durch n und q aus.    



c = p + q 

Höhensatz:

h² = q * p 

h = n * q 

h² = (n * q)² = n² * q² 

n² * q² = q * p | :q 

n² * q = p 

In c = p + q eingesetzt:

c = n² * q + q 

c = q(1 + n²)
